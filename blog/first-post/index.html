我的第一篇技术笔记

2026-06-09心得入门

为什么开始写技术笔记？

作为一名开发者，我一直相信 输出是最好的输入。阅读技术文章、学习新框架之后，如果不及时整理和总结，很多知识很快就会遗忘。

写笔记的好处

  1. 加深理解 — 用自己的话复述一遍，才能真正检验是否理解
  2. 建立知识体系 — 将零散的知识点串联起来
  3. 便于回顾 — 几个月后回过头来看，能快速捡起来

如果你不把知识写下来，它就只是过眼云烟。

技术栈选择

这个网站使用以下技术构建：

  • Next.js 14 — React 全栈框架
  • Tailwind CSS — 实用优先的 CSS 框架
  • TypeScript — 类型安全的 JavaScript
  • Vercel — 零配置部署平台
